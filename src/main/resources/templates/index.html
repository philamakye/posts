About this API

This CRUD API has five endpoints:

  • POST request to create a new post https://posts-production.up.railway.app/posts
  • GET request to view all posts https://posts-production.up.railway.app/posts
  • GET request to view single post https://posts-production.up.railway.app/posts/{id}
  • PUT request to update a post https://posts-production.up.railway.app/posts/{}
  • DELETE request to delete a post https://posts-production.up.railway.app/posts/{}
